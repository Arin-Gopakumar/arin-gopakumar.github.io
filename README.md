# Arin Gopakumar - Personal Website

A beautiful, compact single-page academic portfolio with modern UI design.

## Features

- **Compact Design**: Single-page layout that fits entirely on one screen
- **Two-Column Layout**: Clean sidebar with profile info and main content area
- **Modern UI**: Minimal, information-dense design with professional aesthetics
- **Responsive**: Works on all devices and screen sizes
- **Print-Friendly**: Looks great when printed

## Getting Started

### Option 1: Open Directly
Simply open `index.html` in your web browser.

### Option 2: Local Development Server
```bash
npm install
npm start
```

This will start a local server at `http://localhost:3000`

## Customization

### Update Profile Picture
Replace the avatar URL in `index.html`:
```html
<img src="YOUR_IMAGE_URL" alt="Arin Gopakumar" class="avatar">
```

### Update Content
Edit the HTML content in `index.html` to update:
- Education
- Research experience
- Publications
- Contact information

### Update Styling
Modify `styles.css` to customize:
- Colors
- Fonts
- Spacing
- Layout dimensions

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Modern CSS Grid & Flexbox

## Design Philosophy

The design follows a compact, single-screen approach inspired by academic portfolios, with:
- Tight vertical spacing for maximum information density
- Small font sizes (0.85-0.9rem) for body text
- Minimal whitespace and padding
- Clean typography with no decorative elements
- Professional academic aesthetic
